# 李夫人

```
北方有佳人，绝世而独立。
一顾倾人城，再顾倾人国。
宁不知倾城与倾国，佳人难再得。
```

**学习`AI`相关知识，搞点东西练手，主要功能：图片识谱、指法标记、音频风格迁移。**

> 如果需要编辑谱面建议使用`musescore`专业编辑软件💃💃💃
