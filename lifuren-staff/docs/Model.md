## 师旷

```
音频特征：音高 + 音调 + 音色
实现原理：原始 + 特征 = 目标
```

#### 数据集

原始文件最好使用纯人声音频，可以使用`UVR5`提取纯净人声。

```
train/音频1.mp3
train/音频2.mp3
val/音频1.mp3
val/音频2.mp3
```
