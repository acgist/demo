# JUST-SHARE

不同设备快速无损共享文件

#### CLI说明

`-r/--recv`表示接收，`-s/--send`表示发送，发送后面跟文件列表。也可以不带参数根据提示进行操作。

```
jshare -r
jshare -s file1 file2 file3 ...
```

## 分享协议

#### 协议结构

```
+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
|    version    |     type      | uint8_t
+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
|          data_id...           | uint16_t
+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
|          data_length          | uint16_t
+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
|          data...              | char*
+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
```

#### 协议类型

|协议标识|协议描述|
|--:|:--|
| 1|ACK|
| 2|SYN|
|10|发现广播|
|20|开始发送|
|21|文件数据|
|22|结束发送|
