## 功能

|功能|快捷键|右键支持|
|:--|:--|:--|
|视频关灯（50%）|Ctrl + Shift + V|✓|
|视频关灯（80%）|Alt + Ctrl + Shift + V|✓|
|多关键词搜索|Ctrl + Shift + S|✓|

## 多关键词配置

### 匹配元素

填写匹配标签名称

### 匹配规则

选择查询规则名称

> 匹配规则：使用所有规则

### 规则名称

规则名称

### 规则文本

具体匹配规则（每行一条）

> 支持正则表达式
